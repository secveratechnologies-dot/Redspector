# RedSpecter Security & Compliance Dashboard Monorepo

Welcome to the RedSpecter Security Dashboard. This repository contains both the frontend React client and the backend Express API service in a unified monorepo.

## Project Structure

- **[frontend/](file:///Users/sanikasadre/Downloads/Redspector/frontend)**: Fully-featured responsive client built with React, Tailwind CSS, Lucide icons, and code-splitting.
- **[backend/](file:///Users/sanikasadre/Downloads/Redspector/backend)**: Scalable multi-tenant REST API built with Node.js, Express, PostgreSQL, Redis, and Prisma ORM.
